# 🚀 クイックスタートガイド

YouTubeの配信から10分の切り抜き動画を5分で始められます！

## 📝 必要なもの（5分で準備）

1. **YouTube API Key** 
   - [Google Cloud Console](https://console.cloud.google.com/)でプロジェクト作成
   - YouTube Data API v3を有効化
   - APIキーを作成してコピー

2. **対象チャンネルID**
   - 切り抜きたいYouTubeチャンネルのURL
   - 例: `https://www.youtube.com/channel/UCrzO_hsFW8vLLy8xFBADfqQ`
   - `UC`から始まるIDをコピー

## ⚡ 3つの選択肢

### オプション1: ローカルで試す（開発者向け）

```bash
# 1. リポジトリをクローン
git clone https://github.com/kyo10310415/Create-cut-out-videos.git
cd Create-cut-out-videos

# 2. 環境設定
cp .env.example .env
# .envファイルを編集してAPIキーを設定

# 3. 依存関係をインストール
pip install -r requirements.txt

# 4. FFmpegをインストール（Ubuntu/Debian）
sudo apt-get install ffmpeg

# 5. 実行
python run_processor.py
```

**所要時間**: 10分  
**コスト**: 無料

## 🎯 処理の流れ（自動）

```
1. YouTube配信を検索 (30秒)
   ↓
2. コメント・統計を分析 (1分)
   ↓
3. 見どころを自動検出 (30秒)
   ↓
4. 動画をダウンロード (5-10分)
   ↓
5. セグメントを切り抜き (2-5分)
   ↓
6. ジャンプカット適用 (3-5分)
   ↓
7. 字幕を自動生成 (5-10分)
   ↓
8. 完成！ (合計: 20-40分)
```

## ⚙️ カスタマイズ（.envファイル）

```env
# 見どころの厳しさ（0.5-1.0）
MIN_HIGHLIGHT_SCORE=0.7  # デフォルト
# 0.5: より多くの場面を抽出
# 0.9: 最高の場面のみ抽出

# 目標動画長（秒）
CLIP_DURATION_TARGET=600  # 10分
# 300: 5分動画
# 1200: 20分動画

# ジャンプカット
JUMP_CUT_ENABLED=true
# true: 無音部分を削除（推奨）
# false: 元の間をそのまま使用

# 字幕サイズ
SUBTITLE_FONT_SIZE=48
# 40: 小さめ
# 60: 大きめ
```

## 🔧 よくある質問

**Q: 処理時間はどれくらい？**
A: 1時間の配信で20-40分程度です。

**Q: コストは？**
A: 週1回で$30/月、毎日で$50/月が目安です。

**Q: 複数チャンネルを処理できる？**
A: はい。`.env`の`TARGET_CHANNEL_IDS`にカンマ区切りで追加してください。

**Q: 字幕の言語は？**
A: 日本語がデフォルトです。英語の場合は`language='en'`に変更してください。

**Q: 処理の進捗を確認するには？**
A: Webダッシュボードのログ出力、またはRenderのログタブで確認できます。

**Q: 生成された動画はどこ？**
A: `output/`ディレクトリに保存されます。

**Q: 失敗した場合は？**
A: ログを確認してエラーメッセージをGitHub Issueに報告してください。
